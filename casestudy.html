PokeDex


My favorite project to work on was a functioning PokeDex.

This project used GitHub, PokeAPI, VSCode, and was coded with HTML, JavaScript, and CSS. This project returned a list of the first 151 Pokemon, as well as a card with information about each. This was the first project utilizing APIs, as I had to connect to an external database called PokeAPI to find and return information about each Pokemon. This project was challenging but a lot of fun, and I enjoyed designing it.

To start with, I had to connect the app to an external API, the PokeAPI. I used the fetch function to accomplish this. I created modals to overlay information about each Pokemon. I designed everything about the app, and chose a dreamlike pink and purple color scheme to reflect my own preference for psychic and fairy types. I had several complications throughout the process which ranged in difficulty, from something missing capitalization to entire blocks of code needing to be rewritten. That said, the process of troubleshooting was both fun and rewarding.

In the end, I had created a website that:
Fetched a list of the first 151 Pokemon from an external API
Revealed a modal that contained and displayed information on each Pokemon

This project was one of my favorites because of my love of Pokemon, and watching it all come together was very satisfying.
